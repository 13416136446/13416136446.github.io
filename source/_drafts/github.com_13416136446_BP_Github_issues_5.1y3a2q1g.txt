我的故事: 
在接触 github 之前, 每天在办公室做着重复的工作, 为了能够快点完成工作, 减少重复的沟通,我去找了很多工具, 有 印象笔记/奇妙清单/坚果云等等等等, 还专门将他们统一起来, 用他打造属于我的系统, 可是打通他们很困难, 维护起来也很辛苦,教会其他同事使用更困难.
直到我遇到 github , 发现它更符合人工作的特点, 在 issue 上,我会设定任务,按时完成,将他们变成 wiki 或者 作品, 开始以输出为导向, 创作和输出. 每一次操作都会被记录, 这让我更有责任感, 写字的时候无时无刻不考虑同事们是否容易看懂. 会为别人着想. 而当我想学习新事物的时候, 只需要在 github上搜索, 看到现有的作品都是开源的, 免费的放在那里, 任你取用. 他们的介绍也都是为你着想, 简单明了容易上手. 不仅仅是工具, 而是思维方式, 变得更开放, 更愿意为他人着想, 学习的更快, 创造也变得容易. 不必担心储存的问题, 也不必担心版本控制. 这些 github 都替我想好了. 
而且在这里, 我的思维可以连接到全球的聪明人, 很快找到与我有相同想法的人, 比如, 我想爬取法律词汇, 有人已经做了, 这个人跟我是同好. 
不仅思维自由, 身体也自由了, 因为可以在网上随时工作, 那么, 想在哪里工作都可以.

对我来说, github 是一个根植于全球 web 和人性的大榕树, 树上的居民们, 与自然和人们无障碍的极速交流, 不断产生新的想法, 实践并转化为果子, 这些果子有些落在地上, 供人使用, 也有些供人调用, 随时生成新的果子和枝蔓, 人们顺着向上, 视野越来越开阔, 阳光也越来越充足. 成为树上的居民, 就不再有以前的烦恼.
